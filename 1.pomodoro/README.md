# ポモドーロタイマー - ゲーミフィケーション版

ゲーミフィケーション要素を追加したポモドーロタイマーアプリケーションです。

## 機能

### 1. 経験値（XP）とレベルアップ
- ポモドーロ1回完了ごとに **10 XP** 獲得
- **100 XP** ごとにレベルアップ
- レベルとXPは画面上部に常時表示

### 2. 実績バッジシステム
以下の実績が獲得可能：

- **初めてのポモドーロ**: 最初のポモドーロを完了
- **3日連続**: 3日連続でポモドーロを完了
- **1週間連続**: 7日連続でポモドーロを完了
- **週間10回完了**: 1週間で10回のポモドーロを完了
- **レベル5達成**: レベル5に到達
- **レベル10達成**: レベル10に到達
- **ベテラン**: 合計50回のポモドーロを完了
- **マスター**: 合計100回のポモドーロを完了

### 3. ストリーク（連続日数）
- 毎日ポモドーロを完了することでストリークが継続
- 現在のストリークと最長ストリークを記録
- ストリーク更新時に🔥アイコンで表示

### 4. 統計とグラフ表示
- **週間統計**: 過去7日間の完了数をグラフで表示
- **月間統計**: 過去30日間の完了数を記録
- 完了率と平均集中時間を計算

## 使い方

### アプリケーションの起動

```bash
cd 1.pomodoro
python app.py
```

### メニュー操作

1. **ポモドーロ開始**: 25分の作業タイマーを開始
2. **統計を見る**: 週間グラフ、完了数、ストリークを表示
3. **実績を見る**: 獲得した実績バッジを確認
4. **終了**: アプリケーションを終了

### ポモドーロサイクル

1. **作業時間**: 25分間集中
2. **短い休憩**: 5分間休憩（4セッションごとに15分の長い休憩）
3. 完了するとXPと実績を獲得

## データ保存

ゲーミフィケーションのデータは `gamification_data.json` に保存されます。

### データ構造

```json
{
  "xp": 経験値,
  "level": レベル,
  "total_completed": 合計完了数,
  "current_streak": 現在のストリーク,
  "longest_streak": 最長ストリーク,
  "last_completion_date": 最終完了日,
  "earned_achievements": ["獲得済み実績のID"],
  "completion_history": [
    {"date": "YYYY-MM-DD", "count": 完了数}
  ]
}
```

## 技術仕様

- **言語**: Python 3
- **依存ライブラリ**: 標準ライブラリのみ（追加インストール不要）
- **データ形式**: JSON
- **タイマー**: 
  - 作業時間: 25分
  - 短い休憩: 5分
  - 長い休憩: 15分

## ゲーミフィケーションの効果測定

このアプリケーションは以下の指標でゲーミフィケーションの効果を測定できます：

1. **継続率**: ストリーク日数で測定
2. **完了率**: 週間・月間の完了数で測定
3. **モチベーション**: レベルアップと実績獲得の頻度で測定
4. **集中時間**: 完了数 × 25分で平均集中時間を算出

## カスタマイズ

### タイマー時間の変更

`app.py` の `PomodoroTimer` クラスで時間を調整できます：

```python
WORK_DURATION = 25 * 60  # 作業時間（秒）
SHORT_BREAK = 5 * 60     # 短い休憩（秒）
LONG_BREAK = 15 * 60     # 長い休憩（秒）
```

### XPとレベルの調整

`gamification.py` の `GamificationManager` クラスで調整できます：

```python
XP_PER_LEVEL = 100      # レベルアップに必要なXP
XP_PER_POMODORO = 10    # 1回の完了で得られるXP
```

### 実績の追加

`gamification.py` の `Achievement.ACHIEVEMENTS` に新しい実績を追加できます：

```python
"new_achievement_id": {
    "name": "実績名",
    "description": "説明",
    "condition": lambda stats: 条件式
}
```

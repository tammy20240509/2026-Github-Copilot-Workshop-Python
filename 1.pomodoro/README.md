# Pomodoro Timer アプリケーション

カスタマイズ可能なポモドーロタイマーアプリケーションです。

## 機能

### 基本機能
- ポモドーロテクニックに基づいた作業時間と休憩時間の管理
- タイマーの開始、一時停止、リセット機能
- 視覚的に分かりやすいタイマー表示

### カスタマイズ機能

#### 1. 作業時間の選択（15/25/35/45分）
- 15分: 短い集中セッション向け
- 25分: 標準的なポモドーロセッション
- 35分: より長い集中が必要なタスク向け
- 45分: 深い集中作業向け

#### 2. 休憩時間のカスタム（5/10/15分）
- 5分: 短い休憩
- 10分: 標準的な休憩
- 15分: 長めの休憩

#### 3. テーマ切替
- **ライトテーマ**: 明るい背景で目に優しい
- **ダークテーマ**: 暗い背景で集中しやすい
- **フォーカステーマ**: ミニマルなデザインで最大限の集中

#### 4. サウンド設定
- **開始音**: タイマー開始時のサウンド
- **終了音**: タイマー終了時のサウンド
- **Tick音**: 毎秒のカウントダウン音

各サウンドは個別にオン/オフを切り替え可能です。

## 使い方

### 起動方法

```bash
cd 1.pomodoro
python3 app.py
```

### 基本操作

1. **タイマーの開始**: 「開始」ボタンをクリック
2. **タイマーの一時停止**: 「一時停止」ボタンをクリック
3. **タイマーのリセット**: 「リセット」ボタンをクリック

### カスタマイズ

1. **作業時間の変更**: 設定パネルで希望する作業時間（15/25/35/45分）を選択
2. **休憩時間の変更**: 設定パネルで希望する休憩時間（5/10/15分）を選択
3. **テーマの変更**: 設定パネルでライト/ダーク/フォーカスを選択
4. **サウンドの設定**: 各サウンド（開始音/終了音/Tick音）のチェックボックスでオン/オフを切り替え

### 設定の保存

すべての設定は自動的に `settings.json` ファイルに保存され、次回起動時に復元されます。
